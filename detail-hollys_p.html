Hollys

Desktop renewal website

Problems
  • 전체적으로 올드한 느낌, 균형이 맞지 않는 배치로 원하는 정보를 한눈에 찾기 힘듦
  • 웹표준에 맞지 않는 디자인
  • 대부분이 정적요소로 밋밋한 느낌, 복잡한 네비게이션 교정
Solution
  • 사용자가 주로 이용하는 컨텐츠들을 기준으로 배치함으로써, 사용자가 원하는 정보를 쉽게 찾을 수 있게 함
  • 웹표준에 맞게 코딩하여 접근성과 편리성을 높임
  • 정적인 요소들을 CSS를 활용하여 동적인 효과 추가 적용
Points
  • 메인 슬라이드를 좌우로 이동하는 기능(화살표, 페이지네이션, 스와이프)을 넣어 편의성을 높임
  • 뉴스를 슬라이더 형식으로 변경
  • TOP버튼 스크롤링에 따른 ON/OFF 및 클릭 시 부드럽게 위로 이동
